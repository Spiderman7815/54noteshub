# **App Name**: 54NotesHub

## Core Features:

- Curated Notes: Provide well-organized notes for B.Tech CSE subjects with links to Google Drive for preview and download.
- Subject-wise Organization: Organize notes by subjects: Digital Electronics, Data Structures, and so on.
- Live Search: Implement a real-time search bar to dynamically filter notes by subject name or keyword.
- AdSense Compliance Check: A tool that checks the generated pages for AdSense compliance and makes optimization recommendations.
- Contact Form: A working contact form using EmailJS with fields for name, email, and message.
- Responsive Design: Ensure the website is fully responsive and mobile-friendly.
- Landing Pages: Generates standard pages, and is optimized for fast loading speed and SEO.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to give an academic and trustworthy feel.
- Background color: White (#F5F5F5) for a clean and readable layout.
- Accent color: Dark blue (#003366) for important links and CTAs, providing contrast and guidance.
- Body font: 'PT Sans', sans-serif, for readability.
- Headline font: 'PT Sans', sans-serif, for clear section titles.
- Use simple, professional icons for categories and file types.
- Maintain a clean, minimal layout with clear sections for easy navigation.
- Subtle transitions for search results and page navigation.
